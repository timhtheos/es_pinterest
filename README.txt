
-- SUMMARY --

I have developed this custom module to add Pinterest widget to an existing Easy Social module.


-- REQUIREMENTS --

Module Variable - http://drupal.org/project/variable
A small description in the Module Monday from Lullabot: Variable - http://lb.cm/4M8

Easy Social - https://drupal.org/project/easy_social


-- INSTALLATION --

* Install as usual. 


-- CONFIGURATION --

* Go to /admin/config/content/easysocial and enable Pinterest widget.

Configuration for Easy Social module can be found in its own README.txt file.
